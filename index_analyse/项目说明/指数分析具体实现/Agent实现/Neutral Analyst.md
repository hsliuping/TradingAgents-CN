# Neutral Analyst (中性风险分析师)

## 1. 角色定义
**Neutral Analyst** (在代码中实现为 `neutral_debator.py`) 代表了风险管理委员会中的理性平衡派。它的核心职责是提供一个不偏不倚的视角，权衡潜在的收益与风险。它不仅是中间人，更是理性的锚点，确保决策不会偏向极端的乐观或悲观。

## 2. 核心职责
- **平衡视角**：综合评估上行潜力和下行风险，倡导温和、可持续的策略。
- **全面评估**：考虑更广泛的市场趋势、潜在的经济变化和多元化策略。
- **调和极端**：指出激进派可能忽视的风险，同时也指出保守派可能错失的机会。

## 3. 输入数据
Agent 接收以下核心输入：
1. **交易员/策略顾问决策** (`trader_investment_plan` 或 `strategy_report`)：作为评估基础。
2. **多维度市场报告**：
   - 宏观经济报告 (`macro_report`)
   - 政策分析报告 (`policy_report`)
   - 板块轮动报告 (`sector_report`)
   - 国际新闻报告 (`international_news_report`)
   - 技术分析报告 (`technical_report`)
   - (在个股模式下为：市场研究、情绪、新闻、基本面报告)
3. **辩论历史** (`history`, `neutral_history`)：当前的辩论上下文。
4. **对手观点**：
   - 激进分析师的最后回应 (`current_risky_response`)
   - 保守分析师的最后回应 (`current_safe_response`)

## 4. 处理流程
1. **双向扫描**：分析报告中的利多和利空因素，判断它们是否大致平衡。
2. **构建论点**：
   - 倡导“风险调整后收益最大化”。
   - 提出折中方案，如分批建仓、设置止损、对冲风险等。
3. **生成回复**：调用 LLM 生成客观、中立、逻辑严密的辩论发言。

## 5. 输出格式
输出为结构化的 Markdown 文本，包含：
- **加粗的核心观点**
- 列表形式的论据支撑
- 针对性的平衡分析段落

**示例输出片段**：
> **Neutral Analyst**:
> **建议轻仓试错，同时做好两手准备。**
> 
> 激进派看到了反弹的潜力，保守派看到了估值的压力，两者都有道理。
> 1. **结构性机会**：市场整体虽有压力，但新能源板块确实呈现出明显的资金流入迹象。
> 2. **不确定性仍存**：外部环境尚未完全明朗，全仓出击风险过大。
> 
> 最佳策略是利用底仓博取反弹收益，同时保留现金以应对可能的回调。不要试图预测市场，而要应对市场。

## 6. 在 Workflow 中的位置
- 位于 **Risk Management Layer (风险管理层)**。
- 与 **Risky Analyst** 和 **Safe Analyst** 并行或轮流发言。
- 其输出将被 **Risk Judge** (Risk Manager) 用于最终决策。
