# Social Media Analyst (社交媒体分析师)

**职责**: 作为专业的中国市场社交媒体和投资情绪分析师，分析中国主要财经平台（如雪球、东方财富股吧等）的投资者情绪，监控财经媒体和新闻报道倾向，识别影响股价的热点事件和市场传言，评估散户与机构投资者的观点差异以及政策变化对投资者情绪的影响，最终评估情绪变化对股价的潜在影响。

**核心输出**: 详细的中文分析报告，量化投资者情绪强度（乐观/悲观程度）和情绪变化趋势，评估情绪变化对短期市场反应（1-5天）的影响，分析散户情绪与市场走势的相关性，识别情绪极端点和可能的情绪反转信号，提供基于情绪分析的市场预期和投资建议。报告末尾附上Markdown表格总结关键发现，并包含情绪指数评分（1-10分）、预期价格波动幅度和基于情绪的交易时机建议。

**调用工具/接口**:
- `get_stock_sentiment_unified()` - 获取统一股票情绪数据 (该工具内部自动识别股票类型并调用相应数据源)

**数据获取流程**:
```python
# 1. 获取股票市场信息，自动识别A股/港股，并获取公司名称
market_info = StockUtils.get_market_info(ticker)
company_name = _get_company_name_for_social_media(ticker, market_info)

# 2. 调用统一股票情绪数据工具
sentiment_data = get_stock_sentiment_unified(
    ticker=ticker,
    date=current_date
)
```

**数据处理与分析**:
`Social Media Analyst` 基于 `get_stock_sentiment_unified` 工具获取的统一股票情绪数据，对中国主要财经平台、新闻媒体的情绪数据进行量化分析，识别关键意见领袖（KOL）的观点、热点事件对股价预期的影响以及政策解读和市场预期变化，并评估市场情绪对投资者信心和决策的影响程度。最终生成包含情绪指数评分、预期价格波动幅度和基于情绪的交易时机建议的详细分析报告。

**降级方案**:
由于中国社交媒体API限制，如果数据获取受限或分析失败，`Social Media Analyst` 将提供带有失败原因的简化报告，例如：“情绪分析受限：[原因]，请明确说明并提供替代分析建议。”
